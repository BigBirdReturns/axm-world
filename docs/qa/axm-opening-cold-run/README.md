# AXM opening cold-run artifact — 2026-07-14

This directory captures the production-path browser acceptance for the rebuilt
opening. The operator journey is generated by `e2e/honest-opening.spec.ts` with
fresh site storage. It uses the shipped cartridge importer, Arc-owned founding
and opening law, the real engine transition, World's ledger/save path, and a
full-page reload; no state injection or direct engine fixture is used.

## Required behavioral receipt

1. Enter The First Charter from an empty profile.
2. Choose the non-default `crown-seal` authored opening option.
3. Enter the exact presented `cellar` challenge.
4. Commit all six Arc-founded agents and explicitly spend one authored Contract.
5. Reload before resolution and recover the same challenge, party, and spend.
6. Resolve through `runCycle`, observe the consequence, and inspect ledger entry 1.
7. Reload again and recover the exact receipt.
8. Clear storage and mount Karazhan through the same receiver without code changes.
