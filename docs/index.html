Arc in → playable web app out. axm-world is a renderer that takes any AXM arc — a deterministic ruleset plus a charter — and turns it into a real, populated organization you can run, cycle by cycle, in your browser.

▶ The First Charter — playable now

The brain

A deterministic rules engine — pure, headless functions like runCycle and a seeded PRNG. The renderer never reimplements the rules; it draws whatever the engine returns.

The player

Roster, assignment, drama, base, and reports across cycles — an arc-agnostic UI that renders whatever charter it's handed, not a single hardcoded game.

The cartridge slot

bootstrapOrg(arc) drives the engine against any arc's tiers, roles, and name pool to generate a populated starting org. That's what turns "loads any arc" into "plays any arc."

Same brain, swappable player. The logic stays portable; the presentation stays replaceable.
